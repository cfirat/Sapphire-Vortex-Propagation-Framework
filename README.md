# Validated Sapphire Vortex-Propagation Framework

This repository contains the simulation notebook, processed output tables, and figure files associated with the manuscript:

**A Validated Numerical Framework for Femtosecond Vortex-Beam Nonlinear Propagation in Sapphire**

The study develops and applies an $$x,y,t,z$$ numerical framework for femtosecond vortex-pulse propagation in transparent nonlinear media. The model is first validated using fused silica and then extended to ordinary and extraordinary sapphire.
